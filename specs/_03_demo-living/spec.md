# Feature Specification: Demo — Living Specs

Committed manual-testing fixture: a completed spec whose context records two loaded living specs (`todos`, `notes-ui`) with one folded back (`todos`). In this repository — which deliberately has no living-specs configuration — opening it in the viewer exercises the Activity panel's degraded path: capability names render with the "content unavailable" note. In a workspace where those capabilities resolve, the same context renders their full content.
